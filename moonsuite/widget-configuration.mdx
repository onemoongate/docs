## Widget URL Structure

```
https://web.moongate.one/token-details/{tokenMint}?apiKey={apiKey}
```

### Path Parameters

| Parameter    | Required | Description                        |
| ------------ | -------- | ---------------------------------- |
| `tokenMint`  | Yes      | Solana SPL token mint address      |

### Query Parameters

| Parameter           | Required | Description                                             |
| ------------------- | -------- | ------------------------------------------------------- |
| `apiKey`            | Yes      | Partner API key (used to track swaps and onramps; also accepts `api_key`, `api-key`) |
| `partner`           | No       | Partner project name (used for theming)                 |
| `memeSuite`         | No       | Set to `true` for legacy iframe mode (hides nav UI)     |
| `themeName`         | No       | `light` or `dark` base theme                            |
| `containerColor`    | No       | Override container background color                     |
| `primaryColor`      | No       | Override primary/accent color                           |
| `secondaryColor`    | No       | Override secondary color                                |
| `cardColor`         | No       | Override card background color                          |
| `primaryTextColor`  | No       | Override primary text color                             |
| `secondaryTextColor`| No       | Override secondary text color                           |
| `primaryBtnTextColor`| No     | Override primary button text color                      |
| `borderRadius`      | No       | Override border radius (number, in pixels)              |
| `wgBorderRadius`    | No       | Override widget border radius (number, in pixels)       |

## Theme Configuration

The `MoonSuiteTheme` object controls the widget appearance:

| Property             | Type   | Description                       |
| -------------------- | ------ | --------------------------------- |
| `primaryColor`       | string | Accent color for buttons and links |
| `backgroundColor`    | string | Widget background color            |
| `cardColor`          | string | Card/section background color      |
| `textColor`          | string | Primary text color                 |
| `secondaryTextColor` | string | Muted/secondary text color         |

## Theme Priority

Themes are loaded in the following priority order:

<Steps>
  <Step title="URL Query Parameters (Highest Priority)">
    Any color or styling values passed as URL query parameters take precedence over all other sources.
  </Step>
  <Step title="Partner Theme from API">
    Theme configuration fetched from `/partner/moonsuite-theme/:projectName` (when `partner` is provided).
  </Step>
  <Step title="Default Dark Theme (Fallback)">
    If no custom theme is configured, the default dark theme is applied.
  </Step>
</Steps>

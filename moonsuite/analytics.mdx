## Overview

MoonSuite tracks user interactions through Segment (forwarded to Amplitude). All events are prefixed with `Widget:`.

<img src="/images/moonsuite-analytics.png" alt="Analytics dashboard — swap statistics, volume, KPIs, and transaction history" style={{ width: '100%', maxWidth: '700px', maxHeight: '400px', objectFit: 'contain' }} />

## Tracked Events

| Event                           | Trigger                  |
| ------------------------------- | ------------------------ |
| `Widget: widget_viewed`         | Page loads               |
| `Widget: token_viewed`          | Token details displayed  |
| `Widget: buy_button_clicked`     | Buy button clicked       |
| `Widget: sell_button_clicked`    | Sell button clicked      |
| `Widget: swap_button_clicked`    | Swap button clicked      |
| `Widget: buy_with_fiat_clicked`  | Buy with Fiat clicked    |
| `Widget: checkout_started`      | Checkout initiated       |
| `Widget: checkout_completed`    | Payment succeeded        |
| `Widget: checkout_failed`       | Payment failed/abandoned |

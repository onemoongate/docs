## 🔧 Step 3: Integration Examples

### Next.js Integration (1 minute)

#### 1. Wrap your app with MoonGateProvider

```tsx
// pages/_app.tsx (Pages Router) or app/layout.tsx (App Router)
import { MoonGateProvider } from '@moongate/sdk';
import '@moongate/sdk/styles.css';

export default function App({ Component, pageProps }: AppProps) {
  return (
    <MoonGateProvider apiKey="your-api-key-here">
      <Component {...pageProps} />
    </MoonGateProvider>
  );
}
```

#### 2. Add the connect button to your components

```tsx
// components/WalletConnection.tsx
import { MoongateConnectButton, useWallet } from '@moongate/sdk';

export function WalletConnection() {
  const { connected, publicKey, disconnect } = useWallet();

  return (
    <div className="flex flex-col items-center gap-4">
      <MoongateConnectButton />
      {connected && (
        <div className="text-center">
          <p className="text-sm text-gray-600">
            Connected: {publicKey?.toString().slice(0, 8)}...
          </p>
          <button
            onClick={disconnect}
            className="mt-2 rounded bg-red-500 px-4 py-2 text-white hover:bg-red-600"
          >
            Disconnect
          </button>
        </div>
      )}
    </div>
  );
}
```

### React (Create React App) Integration (1 minute)

#### 1. Wrap your app with MoonGateProvider

```tsx
// src/App.tsx
import { MoongateConnectButton, MoonGateProvider, useWallet } from '@moongate/sdk';
import React from 'react';
import '@moongate/sdk/styles.css';

function WalletComponent() {
  const { connected, publicKey, disconnect } = useWallet();

  return (
    <div className="text-center">
      <MoongateConnectButton />
      {connected && (
        <div className="mt-4">
          <p>Connected: {publicKey?.toString()}</p>
          <button
            onClick={disconnect}
            className="mt-2 rounded bg-red-500 px-4 py-2 text-white"
          >
            Disconnect
          </button>
        </div>
      )}
    </div>
  );
}

function App() {
  return (
    <MoonGateProvider apiKey="your-api-key-here" network="devnet">
      <div className="App">
        <header className="App-header">
          <h1>My Solana React App</h1>
          <WalletComponent />
        </header>
      </div>
    </MoonGateProvider>
  );
}

export default App;
```

## 🔧 Available Hooks & Components

### Hooks
- **useWallet()** - Access wallet state and methods
- **useConnection()** - Access Solana connection  
- **useWalletModal()** - Control wallet modal visibility

### Components
- **MoongateConnectButton** - Main connect button (recommended)
- **WalletModal** - Standalone wallet selection modal
- **WalletPreview** - Inline wallet display without modal
- **WalletConnectButton** - Individual wallet connect button
